E-Learning Module

An application for first year students of Vasavi College of Engineering to learn introductory courses through videos.

## Usage

Clone it:

```
$ git clone 
```

Go into the project directory and run the command:

```
$ npm start
```

Open `http://localhost:3000` and enjoy!
